Не используй \( \) для инлайн-формул, вместо этого используй знак доллара $.

Пример:

$$
y = \sigma(w^T * x + b)
$$

где:
- $y$ - бинарный выход (0 или 1, обозначающий классы);
- $\sigma$ - sigmoid;
- $w$ - вектор весов (коэффициентов) для каждого признака;
- $x$ - вектор признаков объекта;
- $b$ - смещение (bias).


Пример и объяснение кода с комментариями, как описано в пункте 6 инструкций, если применимо и актуально для текстового фрагмента (чанка).

Пример:

```python
def my_function(name: str, age: int) -> str:
    """
    Description:
        Описание функции.

    Args:
        name: Имя пользователя.
        age: Возраст пользователя.

    Returns:
        Строка приветствия с именем и возрастом пользователя.

    Raises:
        TypeError: Если тип аргумента name не str или тип аргумента age не int.

    Examples:
        >>> my_function("Иван", 30)
        'Привет, Иван! Тебе уже 30 лет.'
    """
    if not isinstance(name, str) or not isinstance(age, int):
        raise TypeError("Аргументы name и age должны быть типа str и int соответственно.")
    return f'Привет, {name}! Тебе уже {age} лет.'
```
